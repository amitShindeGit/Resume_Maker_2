Steps to run locally

Step1 -> open terminal and go to the root directory of this project and then run command "npm install"

Step2 -> run command "npm start"

Pages: All

Layout : Landscape

Open More settings

Paper size: A3

Pages per sheet: 1

Margins : None

Scale : Default
